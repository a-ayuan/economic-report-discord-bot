## Setup
1. Create a Discord bot + invite it to your server with permission to post messages.
2. Copy `.env.example` to `.env` and fill in:
   - DISCORD_BOT_TOKEN
   - DISCORD_CHANNEL_ID

3. Install deps:
```bash
pip install -r requirements.txt
